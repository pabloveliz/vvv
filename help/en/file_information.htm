File information

The upper part of this window shows information about the selected file: 

fixed and virtual location; 

file name, date and size;

MP3 metadata (if any);

You can type descriptive information in the lower part of the window, in the Description box. 
Both the OK and Cancel button will close the window. If you changed the description OK will save the changes and Cancel will discard them. 
You can set the description for many files at a time by selecting all the files before selecting the Edit/Object Information... menu in the main program window.
